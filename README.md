# Strona adopcyjna zwierząt

Prosta strona internetowa stworzona w celu zaliczenia przedmiotu.  
Strona umożliwia przeglądanie zwierząt do adopcji (psy i koty) z krótkim opisem oraz lokalizacją na mapie.

## Technologie

- HTML5
- CSS3
- Google Maps Embed API

## Struktura projektu

adopcje-zwierzat/
│
├── index.html # Strona główna z mapą i powitaniem
├── dogs.html # Podstrona z psami do adopcji
├── cats.html # Podstrona z kotami do adopcji
├── style.css # Arkusz stylów CSS
├── images/ # Folder ze zdjęciami zwierząt
│ ├── dog1.jpg
│ ├── dog2.jpg
│ ├── cat1.jpg
│ └── cat2.jpg


## Funkcjonalności

- Menu nawigacyjne między podstronami
- Wyświetlanie psów i kotów do adopcji wraz z opisem i zdjęciem
- Interaktywna mapa Google z lokalizacją adopcji
- Responsywny układ strony dzięki CSS

## Jak uruchomić

1. Sklonuj repozytorium:

```bash
git clone https://github.com/MChmielewski97/Strona-adopcyjna.git
```
2. Otwórz folder projektu w VS Code lub innej przeglądarce.

3. Otwórz index.html:

Najwygodniej z Live Server w VS Code:
Prawy przycisk na index.html → Open with Live Server

Lub dwuklik w plik index.html w Eksploratorze, aby otworzyć w przeglądarce.
